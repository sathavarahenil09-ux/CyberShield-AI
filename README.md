# 🛡️ CyberShield AI

> A Modern Cybersecurity Toolkit for Threat Awareness and Basic Security Analysis

CyberShield AI is a web-based cybersecurity project developed as part of the **GTU SBTP Cyber Security Internship 2026**. The application provides multiple defensive security tools that help users understand common cyber threats, analyze passwords, detect phishing attempts, generate cryptographic hashes, and improve cybersecurity awareness.

Unlike server-based applications, CyberShield AI is a **100% static website** that runs completely inside the browser and can be deployed easily using **GitHub Pages**.

---

# 📌 Project Objective

The objective of CyberShield AI is to provide a beginner-friendly cybersecurity toolkit that demonstrates practical security concepts through an interactive web interface.

The project focuses on:

- Password Security
- Phishing Detection
- URL Analysis
- Hash Generation
- MITRE ATT&CK Awareness
- Security Awareness Training

---

# ✨ Features

## 🔐 Password Strength Checker
- Password strength analysis
- Security score
- Entropy estimation
- Suggestions to improve passwords

---

## 🔑 Password Generator

- Generate strong random passwords
- Adjustable password length
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols

---

## 🌐 URL Security Analyzer

Analyzes URLs for:

- HTTPS support
- Suspicious keywords
- URL shorteners
- Risky file extensions
- Encoded characters
- Possible phishing indicators

---

## 🎣 Phishing Email Detector

Detects common phishing characteristics such as:

- Urgent language
- Suspicious links
- Credential requests
- Attachment warnings
- Social engineering keywords

---

## 🔒 Hash Generator

Generate secure hashes using:

- SHA-1
- SHA-256
- SHA-512

---

## 🛡 MITRE ATT&CK Quick Reference

Provides introductory information about common attacker techniques and defensive awareness.

---

## 🧠 Cyber Security Quiz

Interactive quiz for improving cybersecurity knowledge.

---

## 📊 Scan History

Stores recent analysis locally using Local Storage.

---

## 📄 Threat Report

Generate a simple security report based on analysis results.
